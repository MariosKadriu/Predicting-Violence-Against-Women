# Predicting-Violence-Against-Women
Exploratory Data Analysis 
Data Preprocessing (Imputation of numerical and categorical missing values, Encoding and Scaling of features, PCA, addressing the dataset's Class Imbalance) 
Clustering unlabeled data using KMeans algorithm 
Modelling using machine learning algorithms (Logistic Regression, Decision Trees, Random Forests, KNN, Naive Bayes, MLP, SVM and XGBoost) 
Evaluating using Accuracy, Precision, Recall and F1-Score metrics.
